# Trabalho G2 de Ciência de Dados
### Raisson Silveira de Souza

Esse trabalho visa aplicar os conceitos da Ciência de Dados, como **pré processamento**, **engenharia de atributos** e **análises** de datasets.  
Foram utilizados os modelos **Random Forest**, **Ridge** (regressão linear) e **XGBoost** em conjunto com **Optuna** para a obtenção de melhores resultados com variabilidade dos hiperparâmetros, por fim, o modelo XGBoost foi utilizado com o **NestedCV** para obtenção e validação dos melhores resultados com validações cruzadas de hiperparâmetros.  
Foi criado e utilizado um dataset real de alunos x disciplinas da Antonio Meneghetti Faculdade com a **devida autorização da coordenação** cumprindo a exigência de **anonimização**, **não publicação** e **utilização exclusiva** do mesmo para trabalhos acadêmicos nas cadeiras de Inteligência Artificial 2 e Ciência de Dados.  
**O dataset não está disponível neste repositório.**
